# 🌟 THE-GOD-SYSTEM: Features Summary 🌟

แอปพลิเคชัน **THE-GOD-SYSTEM** เป็นระบบจำลองการใช้ชีวิตในโลกจริงให้กลายเป็นเกมสวมบทบาท (Gamification RPG System) โดยผสานการทำงานของ AI เพื่อทำหน้าที่เป็น "ระบบ" หรือผู้ช่วยส่วนตัวของคุณ

---

## 🎮 1. RPG Progression System (ระบบการพัฒนาตัวละคร)
* **เลเวลและแรงก์ (Level & Rank):** สะสมค่า EXP จากการทำเควสต์เพื่ออัปเลเวล และเลื่อนระดับแรงก์ (E, D, C, B, A, S, SS, SSS)
* **ฉายาอาชีพ (Job Titles):** เมื่อระดับแรงก์สูงขึ้น จะได้รับฉายาใหม่ เช่น ฮันเตอร์มือใหม่, นักสำรวจ, ฮันเตอร์ระดับชาติ ไปจนถึง มอนาร์ค
* **ค่าสถานะ (Stats):** มีระบบสเตตัส 5 ด้าน ได้แก่ STR (พละกำลัง), AGI (ความว่องไว), INT (สติปัญญา), VIT (ความทนทาน), SENSE (ประสาทสัมผัส) ที่จะเติบโตขึ้นตามหมวดหมู่ของเควสต์ที่ทำ
* **ระบบพลังชีวิต (HP System):** มีค่าพลังชีวิต หากทำภารกิจล้มเหลว หรือไม่ทำเควสต์ตามกำหนดเวลา HP จะลดลง

## 📜 2. Quest Management System (ระบบจัดการภารกิจ)
* **เควสต์ประจำวัน (Daily Quests):** ระบบสุ่มสร้างเควสต์ประจำวันให้โดยอัตโนมัติ (เช่น ออกกำลังกาย, อ่านหนังสือ, พัฒนาตัวเอง)
* **เควสต์สร้างเอง (Custom Quests):** ผู้เล่นสามารถสร้างภารกิจของตัวเองได้ โดยระบุหัวข้อ และกำหนดความสำคัญ (Priority) 
* **ภารกิจย่อย (Subtasks):** สามารถซอยย่อยเควสต์ใหญ่ให้กลายเป็น Checklists เล็กๆ เพื่อให้ทำสำเร็จได้ง่ายขึ้น
* **เควสต์ทำซ้ำ (Recurring Quests):** รองรับการตั้งค่าเควสต์แบบทำซ้ำ (เช่น ทำทุกวัน)
* **ระบบเส้นตาย (Deadlines):** กำหนดเวลาสิ้นสุดของเควสต์ เควสต์ที่ไม่มีเส้นตายจะถูกยกยอดไปวันถัดไปโดยอัตโนมัติ

## 🤖 3. AI System Assistant (ระบบผู้ช่วย AI อัจฉริยะ)
* **AI Chat:** คุยกับ "ระบบ" ได้โดยตรงผ่านหน้าต่างแชท AI สามารถจดจำบริบทการสนทนาข้ามเซสชันได้ (Memory Service)
* **AI Quest Evaluation:** เมื่อผู้เล่นสร้างเควสต์เอง AI จะช่วยวิเคราะห์และกำหนดค่า EXP และ Stat ที่จะได้รับให้อย่างสมเหตุสมผลโดยอัตโนมัติ
* **AI Skills Integration:** AI สามารถรันคำสั่ง (Skills) แทรกแซงระบบได้โดยตรง เช่น:
  * วิเคราะห์สเตตัสและให้คำแนะนำ (`analyzeStatsSkill`)
  * สร้างเควสต์พิเศษให้ผู้เล่น (`createQuestSkill`)
  * ฟื้นฟู HP ให้ผู้เล่นเมื่อทำดี (`restoreHpSkill`)
  * สั่งลงโทษหัก HP/EXP เมื่อผู้เล่นอู้ (`applyPenaltySkill`)

## ⚔️ 4. Penalty & Daily Reset System (ระบบลงโทษและตัดวัน)
* **Daily Reset:** ระบบตรวจสอบการขึ้นวันใหม่แบบเบื้องหลัง (Background Checker) หากถึงเวลาที่กำหนด (เช่น ตี 4) ระบบจะเคลียร์เควสต์รายวันและตั้งรอบใหม่
* **Punishment:** หากมีเควสต์ประจำวัน หรือเควสต์ที่หมดเวลา (Expired Deadline) แล้วยังทำไม่เสร็จ ระบบจะลงโทษโดยการหัก HP และ EXP โดยอัตโนมัติเมื่อขึ้นวันใหม่

## 💾 5. Data & Cloud Sync (ระบบบันทึกข้อมูล)
* **Cloud Sync (Firebase):** ซิงค์ข้อมูลข้ามอุปกรณ์แบบ Real-time ข้อมูลตัวละครไม่มีวันหาย
* **Offline Fallback (Local Storage):** หากอินเทอร์เน็ตมีปัญหา ระบบจะยังคงทำงานได้และบันทึกข้อมูลไว้ในเครื่องชั่วคราว
* **Export/Import Save Data:** สามารถ Export ข้อมูลตัวละครออกมาเป็นไฟล์ `.json` และ Import กลับเข้าไปได้

## 🎨 6. User Interface (ส่วนติดต่อผู้ใช้งาน)
* **Cyberpunk / System UI:** ดีไซน์หน้าต่างสไตล์เกม RPG ล้ำยุค โทนสีมืด (Dark Mode) ตัดกับแสงนีออน (Glassmorphism + Glow Effects)
* **System Alerts:** มี Pop-up แจ้งเตือนฉุกเฉินสไตล์เกมเมื่อมีการอัปเลเวล (Level Up) หรือโดนบทลงโทษ (Penalty)
* **Mobile Responsive:** รองรับการแสดงผลบนหน้าจอมือถือได้อย่างสมบูรณ์แบบ
